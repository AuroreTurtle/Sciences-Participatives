Création d'une nouvelle observation


/!\ Avant de remplir le formulaire, assurez-vous que l'espèce observée est bien une espèce présente dans la liste de ce site.

Pour la création d'une observation, il faut :

  1. La localisation en renseignant la ville, le département, le pays.
  2. La date de l'observation.
  3. Une photo de l'observation pour vérifier que l'espèce a été bien identifiée.
  4. Le nombre d'individu de l'espèce observée ou bien une estimation du nombre d'individu s'il y en a trop.
  5. Sélectionner l'espèce vue.
