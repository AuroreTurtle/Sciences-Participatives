Bienvenue sur le site de Sciences Participatives


En fonction de votre situation (avec un compte ou non), vous aurez accès à certaines pages spécifiques

Sans compte

Sur le site vous avez accès sans vous connecter à :

  1. la liste des espèces observables
    2. La liste est triée par catégorie : Plante, Champignon, Insectes, Reptiles/Amphibiens, Mammifères et Oiseaux.
    Vous pourrez ainsi facilement retrouver l'espèce que vous recherchez pour savoir si elle fait partie de nos recherches. Si vous avez trouvé l'espèce que vous recherchiez, vous pouvez cliquer dessus pour avoir plus d'informations.

  2. la liste des observations déjà faites par des utilisateurs
    3. Vous aurez accès aux observations ainsi que toutes les informations de l'observation.

Si vous voulez rentrer une observation, il faut créer un compte ici .


Avec un compte et connecté

En vous connectant, vous allez pouvoir créer une observation c'est-à-dire rentrer toutes les informations de l'espèce observée. Vous allez pouvoir ainsi voir toutes les observations que vous avez rentrées dans votre page d'accueil.
