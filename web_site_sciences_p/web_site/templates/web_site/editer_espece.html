Editer la fiche {{ficheEditer.Nom_vernaculaire}} ({{ficheEditer.Nom_scientifiques}})


Pour modifier la fiche Espèce, il faut comme pour sa création:

  1. Le nom vernaculaire et le nom scientifique.
  2. Les informations sur la classification : Famille et Ordre.
  3. Une photo de l'espèce.
  4. Une description pour avoir des informations sur l'espèce ainsi que des détails d'identification.
  5. Des informations sur la répartition géographique pour savoir où on peut trouver l'espèce.
  6. Des informations sur l'habitat de l'espèce pour connaitre le type d'habitat où l'on peut voir l'espèce.
  7. Le numéro TaxRef pour avoir le lien vers l'INPN de l'espèce si l'utilisateur veut plus d'informations.
  8. Le statut UICN à choisir parmi les catégories suivantes: Eteinte (EX), Eteinte à l’état sauvage (EW), En danger critique (CR), En danger (EN), Vulnérable (VU), Quasi menacée (NT), Préoccupation mineure (LC), Données insuffisantes (DD), Non évaluée (NE).
  9. La catégorie de l'espèce pour retrouver plus facilement l'espèce dans la liste.
